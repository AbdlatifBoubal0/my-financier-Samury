# Screenshots

This folder holds the application screenshots referenced by the main README.

The current images are captured in chronological order (`Capture d'écran 2026-07-20 …`).
The README wires the first few into its Screenshots table in that order, so captions
are indicative rather than exact.

To curate:
- Rename the best shots to clean names (e.g. `dashboard.png`, `wallets.png`, `zakat.png`)
  and update the paths in `../../README.md`. GitHub is case-sensitive.
- Keep each image under ~500 KB for fast loading.
